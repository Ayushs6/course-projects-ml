# Project 3 — MNIST Classification with MLP (Deep Learning)

This project implements and compares multiple Multi-Layer Perceptron (MLP) architectures for MNIST digit classification, analyzing performance across different network designs and training settings.

## Contents
- `MachineLearning.ipynb` — training + evaluation notebook
- `Assignment_3_Report.pdf  — writeup and results

## How to run
Recommended: Google Colab.
1. Open the notebook
2. Run cells top-to-bottom

## Notes
- MNIST is loaded inside the notebook (no external dataset upload required).
- The report summarizes model variations, results, and conclusions.
